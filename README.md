# AppDoc – Clinic Management System

AppDoc is a web-based clinic management system developed using ASP.NET, C#, and SQL Server to streamline clinic operations such as patient management, doctor management, and appointment scheduling.

## Features
- Patient registration and login
- Doctor profile management
- Appointment booking system
- Admin management panel
- Role-based access control

## Technologies Used
- ASP.NET Web Forms
- C#
- SQL Server
- HTML
- CSS

## Project Purpose
This project was developed as part of a university assignment to demonstrate web application development and database integration for healthcare management systems.
